Este código em Python utiliza a biblioteca spaCy para analisar automaticamente um texto em português, especificamente o livro Harry Potter e a Pedra Filosofal, com o objetivo de identificar relações entre personagens com base na sua presença conjunta em frases. Primeiro, é carregado o modelo de língua portuguesa pt_core_news_lg, que permite fazer tarefas de processamento de linguagem natural como segmentação de frases e reconhecimento de entidades nomeadas. De seguida, o ficheiro de texto é aberto e lido na totalidade, sendo depois processado pelo spaCy, o que resulta num objeto que contém informação linguística estruturada.

O programa percorre então cada frase do texto através de doc.sents. Para cada frase, são analisadas as entidades reconhecidas (sent.ents) e filtradas apenas aquelas classificadas como pessoas (com a etiqueta "PER"). Os nomes dessas entidades são guardados numa lista temporária, garantindo que não existem duplicados dentro da mesma frase. Se uma frase contiver mais do que um personagem, o código considera que existe uma relação entre eles.

A partir daí, o script cria pares de personagens que aparecem juntos na mesma frase e regista essas ocorrências num dicionário chamado amigos. Este dicionário funciona como uma estrutura de relações, onde cada personagem está associado a outros personagens e ao número de vezes que aparecem juntos ao longo do texto. Sempre que dois personagens voltam a aparecer na mesma frase, o contador dessa relação é incrementado.

No final, o programa imprime o dicionário completo, que representa uma espécie de rede de interações entre personagens. Este tipo de análise pode ser útil para explorar relações numa narrativa, identificar personagens centrais ou até construir grafos de redes sociais literárias. No entanto, existem algumas limitações, como o facto de diferentes formas do mesmo nome poderem ser tratadas como entidades distintas, ou o reconhecimento de entidades nem sempre ser perfeito.
